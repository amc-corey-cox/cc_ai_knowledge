## About This Project

This knowledge base serves as the RAG source for [cc_forge](https://github.com/amc-corey-cox/cc_forge), a local-first AI coding assistant. Every article includes YAML frontmatter with source attribution, verification status, and confidence levels. See the [provenance design](reference/provenance.md) for details on the trust model.
