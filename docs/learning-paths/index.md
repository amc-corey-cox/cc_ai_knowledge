# Learning Paths

Learning paths provide a guided order through the knowledge base articles. Instead of browsing topics individually, a path builds concepts sequentially — each article lays groundwork for the next.

## Available Paths

### [AI for Developers](ai-for-developers.md)

**Audience:** Experienced developer who uses AI daily but wants stronger fundamentals.

A structured walk through 23 articles across 7 modules. Starts with how transformers and attention actually work, progresses through the models you use and how to prompt them effectively, then covers agent architectures and agentic coding in practice. Optional modules cover multi-agent patterns and running models locally.

**Modules:**

1. How LLMs Actually Work (3 articles)
2. The Models You Use (2 articles)
3. Talking to Models Effectively (1 article)
4. How AI Agents Work (5 articles)
5. Agentic Coding in Practice (5 articles)
6. Extending Capabilities with MCP (2 articles)
7. Running Models Locally (3 articles, optional)

*Articles marked optional within modules can be skipped without losing the thread.*
